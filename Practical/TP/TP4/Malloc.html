Introduction

The purpose of this practical is to write an instructive version of malloc() and its associated functions: free(), calloc() and realloc().

There are different implementations of malloc(). They depend on the operating systems and their architectures. The implementation you are going to write in this practical is a limited approach of real implementations. However, it will give you a deep understanding of how memory is dynamically allocated and freed.

Before writing some code, you need to have some basic knowledge about memory organization.

Memory Organization

Virtual Memory Map

Let us start with the following virtual memory map:

This simplified memory map represents the memory organization of a running program. The lowest address (0) is at the top of the map. The highest address is at the bottom of the map. The size of the addresses (their number of bits) depends on the architecture.

The stack pointer is an address that points to the top of the stack, which is usually used to store local variables.

The program break is an address that points to the end of the heap. It is usually incremented or decremented by the malloc() and free() functions according to the current amount of allocated memory.

Moving the Program Break

To allocate and free a memory space in the heap without calling malloc(), we can increment and decrement the program break by using the sbrk(2) function.

For instance, this program stores the value of an integer in the heap. 

#include <unistd.h>

int main()
{
    // Gets the current program break.
    int* heap = sbrk(0);

    // Increments the program break by the size of an int.
    // (Allocates the memory space in the heap.)
    sbrk(sizeof(int));

    // Stores a value in the heap.
    *heap = 24;

    // Frees the memory space.
    sbrk(-sizeof(int));

    return 0;
}

When the parameter zero is passed to sbrk(), this function returns the value of the current program break. When the parameter is different from zero, it increments or decrements the program break. In that latter case, the return value can be either the previous or the new program break: it depends on the operating system. So, except for the parameter 0 and for error handling, we will never use the return value of sbrk() (for 0, the previous and new program breaks are the same).

For the whole practical, you are not allowed to use malloc(), calloc(), realloc() and free() from the standard library. But be careful, some other functions can modify the program break. So, for this practical, you should call these functions cautiously. For instance, in some operating system, a first call to printf() modifies the program break. It means that printf() allocates some memory space in the heap and keep it available for the next calls.

printf_sbrk.c
#include <stdio.h>
#include <unistd.h>

int main()
{
    void* program_break;

    program_break = sbrk(0);
    printf("program_break = %p\n", program_break);

    program_break = sbrk(0);
    printf("program_break = %p\n", program_break);

    program_break = sbrk(0);
    printf("program_break = %p\n", program_break);

    program_break = sbrk(0);
    printf("program_break = %p\n", program_break);

    return 0;
}

Memory Chunks

To write malloc() and its associated function, you will have to organize the heap into a series of memory chunks. Each chunk will be made up of a header and a data section. The first header will be a sentinel, which does not contain any data.

Here is an example:

The structure of a header is as follows: 

typedef struct header
{
    // Address of the previous header.
    struct header* prev;

    // Address of the next header.
    struct header* next;

    // Size in bytes of the data being used.
    size_t size;

    // YES (1), if the data is free.
    // NO  (0), if the data is not free.
    size_t free;
} header;

Note that the size field does not contain the total amount of data. It contains the number of bytes allocated for the program (the argument passed to malloc()). The total number of bytes in the data section may be larger.

The total number of bytes in a data section is the difference between the starting address of the data section and the next field.

The starting address of a data section is the starting address of a header plus the size in bytes of a header.

For performance and technical reasons, the starting address of a data section must be a multiple of eight (the size in bytes of a pointer). Therefore, the size of a chunk must be a multiple of eight. Note that the size of the header structure is already a multiple of eight. It contains two pointers and two size_t fields (the size of the size_t type is the same as the size of a pointer). Therefore, the total number of bytes for a data section has to be a multiple of eight.

If a chunk is free, the size field is irrelevant and should not be used (you do not have to reset it to zero).

The next field of the last header points to the current program break. Therefore, this last header can be identified by comparing the next field with the return value of sbrk(0).

The malloc.h File

malloc.h
#ifndef MALLOC_H
#define MALLOC_H

// Structure of a header.
typedef struct header
{
    // Address of the previous header.
    struct header* prev;

    // Address of the next header.
    struct header* next;

    // Size of the data being used.
    size_t size;

    // YES (1), if the data is free.
    // NO  (0), if the data is not free.
    size_t free;
} header;

// Size of a header in bytes.
extern const size_t HSIZE;

// Constants used for the "free" field.
extern const size_t YES;
extern const size_t NO;

// Gets the starting address of a data section.
void* get_data(header* h);

// Gets a header from a data pointer.
// The data pointer must point to a valid data section.
header* get_header_from_data(void* data);

// Gets the total number of bytes for a data section.
size_t get_total(header* h);

// Gets the address of the sentinel.
header* get_sentinel();

// Initializes the heap with the sentinel.
void init_heap();

// Expands the heap.
// last_header: Address of the last header.
// size: Number of new bytes to store in the heap.
//       This value must be a multiple of eight.
// Returns the address of the new header.
header* expand_heap(header* last_header, size_t size);

// Finds the first free chunk that can contain
// the number of bytes passed as a parameter.
// Returns the header of that chunk.
// If no chunk can be found, returns the last header.
header* find_free_chunk(size_t size);

// Allocates "size" bytes and returns a pointer to the allocated memory.
void* my_malloc(size_t size);

// Allocates memory for an array of "nmemb" elements of "size" bytes each
// and returns a pointer  to  the  allocated memory.
void* my_calloc(size_t nmemb, size_t size);

// Frees the memory space pointed to by "ptr".
// The data pointer must point to a valid data section.
void my_free(void* ptr);

// Changes the size of the memory block pointed to by "ptr" to "size" bytes.
// The data pointer must point to a valid data section.
void* my_realloc(void* ptr, size_t size);

#endif

The test.c File

test.c
#include <stdio.h>
#include <unistd.h>
#include "malloc.h"

void print_header(header* h)
{
    void* base = get_sentinel();

    if (h->prev == NULL)
        printf("prev          = Null\n");
    else
        printf("prev          = 0x%lx\n", (void*)h->prev - base);

    if (h->next == NULL)
        printf("next          = Null\n");
    else
        printf("next          = 0x%lx\n", (void*)h->next - base);

    printf("size (used)   = 0x%02zx (%lu)\n", h->size, h->size);

    if (h == base)
    {
        printf("size (unused) = No data\n");
        printf("size (total)  = No data\n");
        printf("data (start)  = No data\n");
        printf("data (end)    = No data\n");
    }
    else
    {
        size_t total = get_total(h);
        size_t unused = total - h->size;
        void* data = get_data(h);

        printf("size (unused) = 0x%02lx (%zu)\n", unused, unused);
        printf("size (total)  = 0x%02lx (%zu)\n", total, total);
        printf("data (start)  = 0x%lx\n", data - base);
        printf("data (end)    = 0x%lx\n", data + total - 1 - base);
    }

    if (h->free)
        printf("free          = \033[32m\033[1mYes\033[0m\n");
    else
        printf("free          = \033[31m\033[1mNo\033[0m\n");
}

void print_heap()
{
    header* sentinel = get_sentinel();
    header* end = sbrk(0);

    printf("             HEAP\n");
    printf("============================== Chunk 0 (sentinel)\n");
    print_header(sentinel);

    size_t count = 1;

    for (header* h = sentinel->next; h != end; h = h->next)
    {
        printf("------------------------------ Chunk %zu (0x%lx)\n",
            count++, (void*)h - (void*)sentinel);
        print_header(h);
    }

    printf("==============================\n");
}

void test_init_heap()
{
    init_heap();
    print_heap();
}

void test_my_malloc()
{
    init_heap();

    printf("my_malloc(0) = %p\n", my_malloc(0));

    my_malloc(5);
    void* p = my_malloc(32);
    my_malloc(200);
    print_heap();

    header* h = get_header_from_data(p);
    h->free = YES;
    print_heap();

    my_malloc(18);
    print_heap();
}

void test_my_calloc()
{
    init_heap();

    printf("my_calloc(0, 0) = %p\n", my_calloc(0, 0));
    printf("my_calloc(5, 0) = %p\n", my_calloc(5, 0));
    printf("my_calloc(0, 5) = %p\n", my_calloc(0, 5));
    printf("my_calloc(0x100000000, 0x100000000) = %p\n", my_calloc(0x100000000, 0x100000000));

    char *p1, *p2, *p3;
    header *h1, *h2, *h3;
    size_t i;

    p1 = my_calloc(2, 5);
    p2 = my_calloc(3, 8);
    p3 = my_calloc(100, 2);
    print_heap();

    h1 = get_header_from_data(p1);
    h2 = get_header_from_data(p2);
    h3 = get_header_from_data(p3);
    h1->free = YES;
    h2->free = YES;
    h3->free = YES;
    print_heap();

    for (i = 0; i < h1->size; i++)
        p1[i] = 0xff;
    for (i = 0; i < h2->size; i++)
        p2[i] = 0xff;
    for (i = 0; i < h3->size; i++)
        p3[i] = 0xff;

    p1 = my_calloc(1, 5);
    p2 = my_calloc(2, 8);
    p3 = my_calloc(1, 2);
    print_heap();

    for (i = 0; i < h1->size; i++)
        if (p1[i] != 0)
            break;
    if (i == h1->size)
        printf("Chunk 1 contains only zeros.\n");
    else
        printf("Chunk 1 does not contain only zeros.\n");

    for (i = 0; i < h2->size; i++)
        if (p2[i] != 0)
            break;
    if (i == h2->size)
        printf("Chunk 2 contains only zeros.\n");
    else
        printf("Chunk 2 does not contain only zeros.\n");

    for (i = 0; i < h3->size; i++)
        if (p3[i] != 0)
            break;
    if (i == h3->size)
        printf("Chunk 3 contains only zeros.\n");
    else
        printf("Chunk 3 does not contain only zeros.\n");
}

Improving my_free()

Change your my_free() function so that:

  • The chunk that must be freed merges with the previous chunk (if free).
  • The chunk that must be freed merges with the next chunk (if free).
  • The program break is decremented when the last chunk is freed.

Here is a new test function that can be used to test your new my_free() function.
