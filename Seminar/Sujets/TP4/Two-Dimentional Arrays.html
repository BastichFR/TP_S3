Matrices as One-Dimensional Arrays

Introduction

The purpose of this part is to introduce good practices on matrix manipulations.

There are many ways to represent matrices in the C language. Matrices are two-dimensional arrays, but experience has shown that in most cases, it is preferable to consider matrices as single-dimensional arrays. This way, matrices are stored in contiguous memory space, thus improving performances. Also, the syntax of one-dimensional arrays is more readable.

To explain how it works, let us consider a 3x3 matrix:

This matrix can be stored in a single-dimensional array as follows:

Accessing a cell can be done by converting the two indices of the two-dimensional array into only one index.

array[i][j] = array[i * cols + j]

  • i is the row number of the cell we want to access.
  • j is the column number of the cell we want to access.
  • cols is the total number of columns.

This method is known as row-major order.

The Square Enumeration

The following enumeration is defined in the test.h file. 

enum Square
{
    EMPTY = '.',
    CHESSMAN = 'O',
    LEGAL_MOVE = 'X'
};

That enumeration must be used to set the value of a square. The elements in the board variable are of char type, which is an integer. So, you can assign the enumeration values to any element of the board. For example: 

board[0][0] = EMPTY;        // Empties the square (0, 0).
board[0][1] = CHESSMAN;     // Assigns a chessman to the square (0, 1).
board[0][2] = LEGAL_MOVE;   // Assigns a legal move to the square (0, 2).

Do not assign any other values to a square.

The Main Function

The main function is given. It creates the board variable (note that it is not initialized). Then, it parses an argument passed to the executable file. Such argument is made up of three characters only:

  • The first character represents the chessman.
    • 'r' or 'R' for the rook.
    • 'b' or 'B' for the bishop.
    • 'q' or 'Q' for the queen.
  • The second character is the row number of the chessman position (from 0 to 7).
  • The third character is the column number of the chessman position (from 0 to 7).

Finally, the main function calls either rook_moves(), bishop_moves() or queen_moves() according to the parameter and prints the result.
